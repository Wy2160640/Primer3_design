# Primer3_design


## 根据rs_number设计引物

python rs_primer.py -r rs4880,rs4888    # 一个或多个rs_number，分别以英文"<strong>,</strong>"隔开

输出结果在脚本所在路径文件夹(注意每次运行，清空reslusts.doc和primers.csv文件)

fasta文件名：   results.doc

primers文件名: primers.csv

本脚本只在python3.5+以上运行过
